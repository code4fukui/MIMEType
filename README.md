# MIMEType

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A collection of common MIME types for various file extensions, available as a JavaScript module.

## Usage

```js
import { MIME_TYPE } from "https://code4fukui.github.io/MIMEType/MIME_TYPE.js";

console.log(MIME_TYPE["html"]);
```

## Data / API

The MIME type mappings are sourced from the official IANA media types registry:

[IANA Media Types](https://www.iana.org/assignments/media-types/media-types.xhtml)
